Examples

This section contains various examples of interactive usage of Pusa. All examples do not use any custom JS code.

/* no custom js */

Content Modification

This example demonstrates the ability to request and dynamically change content.

Clipboard

Select any text on the page, click copy and paste. The marked text will be copied to the clipboard, sent to the backend, and after processing placed into the textarea.

Cursor Coordinates

Move the cursor over the field. The backend handles the mousemove event and outputs the coordinates. In this example, the update frequency is limited to 800 milliseconds.
