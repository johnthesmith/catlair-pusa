Pusa uses a declarative format of directives that are generated during backend method execution.

[ [ "action", "helloworld-click", [ [ "alert", "Hello world" ] ] ], [ "body" ], [ "children", [ "equal", "#id", "helloworld" ] ], [ "event", "click", "helloworld-click" ] ]

Thus, the development of the active part of the frontend application has been moved to the backend.
