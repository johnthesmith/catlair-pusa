Examples

Раздел содержит различные примеры интерактивного использования Pusa. Все примеры не используют специфический js код, но исключительно библиотеку framework pusa.js.

Backend код примеров размещен в файле pusa_pages.php.

/* no custom js */

#1 Изменение контента

Пример демонстрирует возможность запроса и динамического изменения контента.

#2 Буффер обмена

Выделите любой текст на странице, нажмите скопировать и вставить. Помеченный текст будет скопирован в буфер обмена, отправлен на backend, и после обработки размещен в textarea.

#3 Координаты курсора

Перемещайте курсон на поле. Backend будет обрабатывать каждое событие mousemove с учетом дискретности и выводить координаты.

500 mls

#4 Forms

Заполните форму и отправьте ее на обработку. Backend выполнит работу и добавит форму в таблицу. В данном примере backend не сохранят данные.

#5 Фокус элементов

Показаны способы позиционирования фокуса на элементах для последующих манипуляций.

#6 Валидация ввода

Пример показывает возможности валидации ввода.

#7 Скроллинг

Демонстрируется возможость позиционирования элемента на странице. Клик переместит страницу на элемент uuid.

#8 Динамическая подгрузка

Демонстрация возможности динамический загрузки контента при скролировании. Изображения предоставляются ресурсом https://picsum.photos

#9 Таймер

Пример даёт возможность запускать и останавливать таймер. Каждый вызов таймера выполняет следующую итерацию с бэкендом, изменяя состояние визуального глюкала.
