Hello world

При запросе этой страницы backend Pusa формирует в том числе следующий HTML код:

... <button id="helloworld"> Нажми меня </button> ...

... а так же выполняет PHP скрипт:

/* Pusa SDK */ $this -> action /* определили новое действие */ ( 'helloworld-click', /* id действия */ [ $this -> copy() /* создали ветку директив */ -> alert( 'Hello World' ) /* указали директиву alert */ ] ) -> body() /* установили фокус на body */ -> childrenById( 'helloworld' ) /* сместили фокус на элемент по id */ -> event( 'click', 'helloworld-click' ); /* назначили действие на событие */

Frontend не получает какого либо специфического JS скрипта для прикладной задачи.

/* без специфического кода */

Обработка активныйх действий выполняется скриптом pusa.js на основе декларативного формата, формируемого сервером.

[ [ "action", "helloworld-click", [ [ "alert", "Hello world" ] ] ], [ "body" ], [ "children", [ "equal", "#id", "helloworld" ] ], [ "event", "click", "helloworld-click" ] ]

Таким образом разработка активной части frontend приложения перенесена на backend.
